## Design System

### Color System
- Primary: #050505 (Void Black)
- Text: #E0E0E0 (Mist White)
- Interactive: #C0C0C0 (Mercury Silver)
- Accent: #D4D4D4 (Liquid Silver)

### Typography
- Headers: Space Grotesk (300-700 weights)
- Body: JetBrains Mono (300-600 weights)

### Animations
- Liquid shine effect (gradient animation)
- Ripple effects on hover
- Smooth transitions (300-700ms)
- Scale and translate transforms
- Opacity fades
- Background position shifts

### Effects
- Glassmorphism with 40px blur
- Box shadows with glow
- Gradient backgrounds
- Mix-blend-mode for cursor
- Border animations
